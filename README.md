# Personal Developer Portfolio

A fully responsive personal developer portfolio built to showcase my full-stack web applications, hardware engineering builds, and competitive programming progress. 

**🔴 Live Demo:** [https://florax5663.github.io/my-first-project/]

## 🛠️ Technologies Used
* **HTML5:** Semantic structuring and layout.
* **CSS3:** Custom styling, Flexbox/Grid layouts, and responsive media queries.
* **FontAwesome:** Scalable vector icons for UI elements.
* **GitHub Pages:** CI/CD deployment and hosting.

## ✨ Featured Projects Highlighted
* **Assignment Submission Portal:** A role-based Flask & MySQL web application with OTP verification for secure academic submissions.
* **Gesture-Controlled Robot:** An Arduino-based hardware build utilizing MPU6050 accelerometers and nRF24L01 transceivers for wireless control.
* **Algorithmic Problem Solving:** Consistent tracking of Data Structures and Algorithms solutions via LeetCode.

## 🚀 How to Run Locally
Since this is a static site, no complex build tools are required to view the code locally.
1. Clone the repository: `git clone https://github.com/Florax5663/GitGitubWorkShop-florax.git`
2. Open the directory and launch `index.html` in any modern web browser.
